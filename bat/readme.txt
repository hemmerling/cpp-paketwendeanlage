Damit im Krohn und Stiller das 
Hochsprachen-"Debugging" funktioniert
muss im C Quellcode die Anweisung

#pragma DEBUG OBJECTEXTEND CODE

stehen.Im K&S ladet man das Programm mit

L test /J

wenn das Programm 'test.c' heisst.
Der Hochsprachen-"Debugger" wird im K&S
mit der Anweisung 'HLL' ein- und wieder
ausgeschaltet.
